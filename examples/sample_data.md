# Sample data

`demo_pool.csv` is synthetic customer-support text classified into five categories: billing, access, payment, technical, and security. It contains deliberately noisy phrasing and a small security category. `label` is present only for the initial labeled set; `ground_truth` exists solely to inspect the synthetic demo and is never consumed by Label Loom.

The data is generated for this repository, contains no customer data, and is MIT-licensed with the project.
